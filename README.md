# CENG420
CENG420 Artificial Intelligence Project - Summer 2017

# Dependencies

Python 3.6

# How to Run

Use randomized board:
```
python project.py -r 
```
Use hard coded file values:
```
python project.py -f
```
Optional Flag to set the number of players
```
-p P:int
```
